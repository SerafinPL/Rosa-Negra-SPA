## Run Locally

Clone the project

```bash
  git clone https://github.com/SerafinPL/Rosa-Negra-SPA.git
```

Go to the project directory

```bash
  cd Rosa-Negra-SPA
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
